Introduction

CipherBridge is a cloud infrastructure platform that enables applications to securely connect to and manage MetaTrader 5 accounts at scale. It handles the complexity of MT5 deployment, credential management, and real-time communication—so you can focus on building your trading platform.

Whether you're building a copy trading platform, trading academy, or automated trading system, CipherBridge provides the infrastructure layer you need.

Who is this for? Developers building trading platforms, fintech startups, and trading academies who need to integrate MT5 without managing infrastructure.

Quick Start

Get started in 5 minutes. Here's how:

Step 1
Request API Access
Contact us at hello@cipherbridge.cloud to request API credentials. We'll provide your API key and account ID within 24 hours.
Step 2
Create Your First Account
Use your API key to create a new MT5 account connection.
curl -X POST https://api.cipherbridge.cloud/v1/accounts \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "mt5_login": "123456",
    "mt5_password": "your_password",
    "mt5_server": "ICMarkets-Live"
  }'
Step 3
Place Your First Order
Once the account is active, you can start trading.
curl -X POST https://api.cipherbridge.cloud/v1/trade \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "account_id": "acc_123abc",
    "command": {
      "type": "PlaceOrder",
      "symbol": "EURUSD",
      "side": "buy",
      "order_type": "market",
      "volume": 0.1
    }
  }'

Authentication

All API requests require authentication using your API key. Include your API key in the Authorization header:

Authorization: Bearer YOUR_API_KEY

API keys are provided when you request access. Keep your API key secure—it provides full access to your accounts.

Important: Never expose your API key in client-side code. Always make requests from your backend server.

API Endpoints

Base URL: https://api.cipherbridge.cloud/v1

Create Account

POST /accounts
Create a new MT5 account connection
Parameters: mt5_login (string), mt5_password (string), mt5_server (string)

Place Order

POST /trade
Execute a trading command
Parameters: account_id (string), command (object)

Get Account Info

GET /accounts/{account_id}
Retrieve account status and balance

Get Positions

GET /accounts/{account_id}/positions
List all open positions

Close Position

POST /accounts/{account_id}/positions/{ticket}/close
Close an open position

Modify Order

PUT /accounts/{account_id}/orders/{ticket}
Modify SL/TP on an existing position or order

WebSocket (Real-Time)

Connect to our WebSocket endpoint for real-time market data and trade updates.

wss://ws.cipherbridge.cloud/v1

After connecting, authenticate by sending your API key:

{"type": "auth", "api_key": "YOUR_API_KEY"}

Subscribe to market data for a specific account and symbol:

{"type": "subscribe", "account_id": "acc_123abc", "symbol": "EURUSD"}

You'll receive real-time ticks:

{
  "type": "tick",
  "symbol": "EURUSD",
  "bid": 1.1000,
  "ask": 1.1002,
  "last": 1.1001,
  "volume": 100,
  "time": 1700000000000
}

Account Lifecycle

Each MT5 account goes through these states:

StateDescription
CREATEDAccount created, awaiting connection
CONNECTINGBridge connecting to MT5 broker
ACTIVEConnected and ready for trading
DISCONNECTEDConnection lost (auto-reconnects)
FAILEDLogin failed—check credentials

Security Overview

Security is built into every layer of CipherBridge:

  • 🔐 Encrypted credentials — MT5 credentials are encrypted at rest
  • 🔒 Secure connections — All communication over TLS 1.2+
  • 🎫 Token-based authentication — API keys for secure access
  • 📦 Isolated environments — Each account runs in isolation

FAQ

Do I need MT5 installed?
No. CipherBridge manages MT5 instances in the cloud. You only need your MT5 credentials.
How fast is execution?
Orders typically execute in under 50ms from API call to broker confirmation.
Can I scale to thousands of users?
Yes. CipherBridge is built for scale—each user gets an isolated environment, and the platform automatically handles capacity.
What happens if my account disconnects?
CipherBridge automatically reconnects. Your API requests will queue until the connection is restored.
How do I get support?
Email us at hello@cipherbridge.cloud for technical support.
